# Dev Story: RendererOpenGL33 — кастомный рендерер RmlUi

Документация полного цикла разработки шаблонного OpenGL 3.3 рендерера для RmlUi.

## Цепочка этапов

```
01 Initial Request → 02 MVP Implementation → 03 Verification → 04 Remediation
```

## Файлы

| # | Файл | Этап | Описание |
|---|------|------|----------|
| 01 | [01-initial-request.md](01-initial-request.md) | **Запрос** | Исходная постановка задачи: портирование референсного рендерера на реентерабельный шаблонный класс со скрытием реализации в `.inl`, explicit instantiation и концептом для GL-контекста. |
| 02 | [02-mvp-implementation.md](02-mvp-implementation.md) | **MVP** | Поэтапный план реализации (A–F) и devlog: каркас данных, геометрия, текстуры, scissor/transform, заглушки, стабилизация архитектуры. Включает чеклист статуса и технические замечания. |
| 03 | [03-verification.md](03-verification.md) | **Верификация** | Независимый аудит кода: сравнение с референсной реализацией `RenderInterface_GL3`, выявление критических багов, проблем производительности, архитектурная оценка. |
| 04 | [04-remediation.md](04-remediation.md) | **Исправления** | Детальное описание всех правок по замечаниям верификации: 2 критических, 3 существенных, 6 стилистических. Сборка подтверждена. |
